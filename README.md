# Скрипт для анонимизации данных

## Установка

```bash
pip install -e .
```

## Использование

### Запуск скрипта

```bash
python -m pyanonymizer Пример.xlsx result.xlsx -c 4
```

### Вывести описание параметров

```bash
python -m pyanonymizer -h
```

## Примечание

Работает не на 100%. Рекомендую проверять результаты глазами.
